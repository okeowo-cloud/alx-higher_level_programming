=======================
`1-my_list Test Module`
=======================

=======
Usage
=======

    >>> MyList = __import__("1-my_list").MyList


    Case:
    Sort list of positive integers
    >>> myList = MyList()
    >>> myList.append(1)
    >>> myList.append(4)
    >>> myList.append(2)
    >>> myList.append(3)
    >>> myList.append(5)
    >>> myList.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(myList)
    [1, 4, 2, 3, 5]

    Case:
    Sort list with negative integers
    >>> myList = MyList()
    >>> myList.append(1)
    >>> myList.append(2)
    >>> myList.append(-6)
    >>> myList.append(4)
    >>> myList.append(0)
    >>> myList.print_sorted()
    [-6, 0, 1, 2, 4]
    >>> print(myList)
    [1, 2, -6, 4, 0]

    Case:
    Sort list with repeating integers
    >>> myList = MyList()
    >>> myList.append(1)
    >>> myList.append(3)
    >>> myList.append(2)
    >>> myList.append(1)
    >>> myList.append(2)
    >>> myList.print_sorted()
    [1, 1, 2, 2, 3]
    >>> print(myList)
    [1, 3, 2, 1, 2]


    case:
    Sort list with all negative numbers
    >>> myList = MyList()
    >>> myList.append(-10)
    >>> myList.append(-1)
    >>> myList.append(-3)
    >>> myList.append(-7)
    >>> myList.append(-5)
    >>> myList.print_sorted()
    [-10, -7, -5, -3, -1]
    >>> print(myList)
    [-10, -1, -3, -7, -5]
